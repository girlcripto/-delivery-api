# Delivery API

Projeto de API REST para um sistema de delivery desenvolvido com Spring Boot.

## Tecnologias

- Java
- Spring Boot
- Maven
- Git

## Como Iniciar

```bash
mvn clean install
mvn spring-boot:run
```

## Estrutura do Projeto

```
src/
├── main/
│   ├── java/
│   │   └── com/deliverytech/delivery_api/
│   └── resources/
│       ├── application.properties
│       ├── static/
│       └── templates/
└── test/
    └── java/
```
